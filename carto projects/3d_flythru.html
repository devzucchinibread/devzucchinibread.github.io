3D Flythrough Animation

This flythrough was built by combining LiDAR, point cloud data, and drone imagery to reconstruct the landscape. Using PLS-CADD exports converted from KMZ to feature classes, I generated multipoint feature classes for poles and wires, giving them full 3D modeling capability.
