Batch Modify Relates Tool (ArcPy)

This ArcPy tool automates the process of maintaining relationships between GIS features and related tables in bulk. While this demo shows its use on a Work Order History (WOH) table, the script is fully adaptable to any dataset with shared key fields.

Example Code:


    with arcpy.da.InsertCursor(rel_class, keys) as cursor:
        try:
            cursor.insertRow((woh_globalid,feature_guid))
            arcpy.AddMessage("Related WOH {} to {}".format(woh_globalid, feature_guid))
        except Exception as e:
            arcpy.AddMessage("Insert Failed: {}".format(e))
            raise
